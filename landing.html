Community Captioner

Free, open-source live captions for OBS. Speech recognition runs locally on your machine for privacy and low latency.

1 Download the App

Download the Community Captioner app and extract it to any folder on your computer.

Download for Mac/Windows/Linux

2 Open Terminal

How to open Terminal on Mac:

  1. Press Cmd + Space to open Spotlight
  2. Type Terminal and press Enter

How to open Command Prompt on Windows:

  1. Press Win + R to open Run
  2. Type cmd and press Enter

How to open Terminal on Linux:

  1. Press Ctrl + Alt + T
  2. Or search for "Terminal" in your applications menu

3 Install Dependencies

Copy and paste these commands into your terminal, one at a time:

First, install portaudio (required for microphone access):

Click to copy brew install portaudio

Then install the Python packages:

Click to copy pip3 install SpeechRecognition pyaudio

Don't have Homebrew?

Install it first by pasting this command:

Click to copy /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Install the Python packages:

Click to copy pip install SpeechRecognition pyaudio

If pyaudio fails to install

Download the wheel file from here and install it with: pip install [downloaded-file].whl

First, install portaudio:

Click to copy sudo apt-get install portaudio19-dev python3-pyaudio

Then install the Python packages:

Click to copy pip3 install SpeechRecognition pyaudio

4 Run the App

After downloading, the zip file is in your Downloads folder. Copy and paste these commands:

Go to your Downloads folder and unzip:

Click to copy cd ~/Downloads && unzip community-captioner.zip

Go into the folder and start the app:

Click to copy cd ~/Downloads/community-captioner && python3 start-server.py

Go to your Downloads folder:

Click to copy cd %USERPROFILE%\Downloads\community-captioner

Start the app:

Click to copy python start-server.py

Go to your Downloads folder and unzip:

Click to copy cd ~/Downloads && unzip community-captioner.zip

Go into the folder and start the app:

Click to copy cd ~/Downloads/community-captioner && python3 start-server.py

What happens next?

Your browser will automatically open to http://localhost:8080 - this is your control panel!

In the control panel, the Remote Server field should already be set to sync with our cloud. Just click "Start Listening" to begin!

5 Add to OBS

In OBS Studio, add a Browser Source with these settings:

üìê Width: 1920

Start speaking and your captions will appear in OBS. Use the local control panel to customize fonts, colors, position, and add a logo overlay.
