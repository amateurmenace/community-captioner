Community Captioner

Free, open-source live captions for OBS. Speech recognition runs locally on your machine for privacy and low latency.

1 Download the App

Download the Community Captioner app and extract it to any folder on your computer.

2 Install Dependencies

Open Terminal (Mac/Linux) or Command Prompt (Windows) and run:

# Install required Python packages
pip3 install SpeechRecognition pyaudio

Mac Users

If pyaudio fails to install, first run: brew install portaudio

3 Run the App

Navigate to the extracted folder and run the start script:

# Navigate to the folder
cd community-captioner

# Start the captioner
python3 start-server.py

This will open the control panel in your browser at http://localhost:8080

Connect to Cloud Server

In the control panel, enter this URL in the "Remote Server" field to sync captions to your OBS overlay:

http://138.197.15.64:8080

4 Add to OBS

Start speaking and your captions will appear in OBS. Use the local control panel to customize fonts, colors, position, and add a logo overlay.
